# CODEFINDER
All-in-one OCR, analysis, and geometric/cipher exploration pipeline.
- Backend: FastAPI + SQLAlchemy + Alembic
- OCR: Tesseract (via pytesseract)
- Frontend: React
- CI: GitHub Actions (pytest + alembic)
